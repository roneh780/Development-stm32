快速开始

1. 本仓库已包含一个简单的静态网站骨架。你可以直接在 GitHub 仓库设置中启用 GitHub Pages：
   - 设置 -> Pages -> Source 选择 `main` branch 下的 `/ (root)` 或 `docs/` 文件夹。
   - 推荐：将 `index.html` 保持在仓库根，如果想使用 `docs/` 发布，可把 index.html 移到 docs/ 或将 Pages 指向根目录中的 `main`。
