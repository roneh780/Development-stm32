目标

  • 安装工具链（STM32CubeIDE / arm-none-eabi / OpenOCD 等）
  • 创建第一个 STM32 项目并点亮 LED

步骤概览

  1. 安装 STM32CubeIDE 或 使用 Make + toolchain
  2. 新建工程，选择你的 STM32 型号
  3. 配置时钟、GPIO，编写点亮 LED 的 main.c

示例代码片段

// 简单 GPIO 点亮示例（伪代码）
#include "stm32f1xx.h"

int main(void){
  // 使能 GPIO 时钟
  RCC->APB2ENR |= (1<<2); // 根据具体系列修改
  // 配置 PA5 为推挽输出
  GPIOA->CRL &= ~(0xF << (5*4));
  GPIOA->CRL |= (0x1 << (5*4));
  while(1){
    GPIOA->ODR |= (1<<5); // LED ON
    for(volatile int i=0;i<100000;i++);
    GPIOA->ODR &= ~(1<<5); // LED OFF
    for(volatile int i=0;i<100000;i++);
  }
}
